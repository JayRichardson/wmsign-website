# Handleiding: GitHub Desktop instellen voor WM Sign

Met GitHub Desktop deploy je de website met een paar klikken — geen Terminal nodig.

---

## Stap 1 — Download en installeer GitHub Desktop

👉 **[desktop.github.com](https://desktop.github.com)**

Klik op **Download for macOS**, open het gedownloade bestand en sleep GitHub Desktop naar je Programma's map.

---

## Stap 2 — Inloggen

1. Open GitHub Desktop
2. Klik **Sign in to GitHub.com**
3. Log in met je GitHub account (JayRichardson)
4. Klik op **Authorize desktop** in de browser die opent
5. Terug in de app: klik **Finish**

---

## Stap 3 — De wmsign-website repository koppelen

1. Klik op **Clone a Repository from the Internet...**
2. Kies het tabblad **GitHub.com**
3. Je ziet `JayRichardson/wmsign-website` in de lijst — klik erop
4. Bij **Local Path**: kies de map `/Users/j.m.richardson/Documents/Claude/Projects/WM Sign`
   - Klik **Choose...** en navigeer naar die map
5. Klik **Clone**

GitHub Desktop downloadt de repository naar je computer.

---

## Stap 4 — Wijzigingen deployen (dit doe je elke keer)

Nadat ik (Claude) bestanden heb aangepast:

1. Open **GitHub Desktop**
2. Links zie je alle gewijzigde bestanden — vink ze aan als ze nog niet aangevinkt zijn
3. Onderaan links: typ een korte beschrijving, bijv. *"Nieuwe foto toegevoegd"*
4. Klik **Commit to main**
5. Klik rechtsboven op **Push origin**

✅ Na ongeveer 1 minuut zijn de wijzigingen live op wmsign.nl
